# pensamento_computacional

Foi feita a leitura de três pinos analógicos (A5, A4 e A3), para que três potenciômetros fossem responsáveis por ajustar a cor de um LED RGB. Foram definidos os pinos(11, 10 e 9) para o LED, logo após foram conectados as variáveis (pote_5, pote_4 e pote_3), para que os potenciômetros tivessem a capacidade de alterar a intencidade do RGB.
